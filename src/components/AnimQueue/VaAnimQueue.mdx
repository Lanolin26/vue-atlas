# Animation queue

Компонент, который позволяет вам определить набор элементов для создания переходов "Вход" и "выход".

AnimQueue - это оболочка для элементов. Вы захотите использовать AnimQueue, когда захотите последовательно анимировать коллекцию элементов.

Важно помнить, что для каждого элемента в коллекции, который нужно анимировать, потребуется уникальный атрибут anim-key.
Кроме того, в большинстве случаев вы, вероятно, захотите убедиться, что каждый элемент в коллекции имеет начальную видимость hidden.
